# Politique de Sécurité

Ce projet respecte les pratiques DevOps :
- **Aucun secret** n'est stocké dans le code source.
- Les identifiants (SSH, Base de données, JWT) sont gérés via **GitHub Secrets**.
- Ils sont injectés comme variables d'environnement au moment du déploiement.

## Rapports de vulnérabilité
Veuillez signaler toute faille de sécurité en ouvrant une issue privée sur ce dépôt.
